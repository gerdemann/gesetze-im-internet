# Markdown Version von Gesetze im Internet

Dieses Repository enthält eine Markdown-Version der Gesetze und Verordnungen von [Gesetze im Internet](https://www.gesetze-im-internet.de/), einer offiziellen Plattform des Bundesministeriums der Justiz und für Verbraucherschutz.

## Über das Projekt

Dieses Projekt stellt deutsche Gesetze und Verordnungen in einem strukturierten Markdown-Format zur Verfügung, um die Nutzung, Suche und Weiterverarbeitung der rechtlichen Texte zu erleichtern.

## Verwendung

Die Markdown-Dateien können direkt gelesen, durchsucht oder in andere Formate konvertiert werden. Jede Datei entspricht einem Gesetz oder einer Verordnung.

## Datenquelle

Die Daten stammen von [Gesetze im Internet](https://www.gesetze-im-internet.de/), einer offiziellen Quelle des Bundesministeriums der Justiz und für Verbraucherschutz.

**Hinweis**: Dieses Repository enthält eine Kopie der Daten in Markdown-Format. Für rechtsverbindliche Auskünfte konsultieren Sie bitte die offiziellen Quellen.

**Rechtliche Situation der Gesetzestexte:**

Die Gesetzestexte selbst sind nach §5 UrhG (Urheberrechtsgesetz) als amtliche Werke zur amtlichen Bekanntmachung **nicht urheberrechtlich geschützt**. Laut [gesetze-im-internet.de](https://www.gesetze-im-internet.de/) stehen die Rechtsnormen "in allen angebotenen Formaten zur **freien Nutzung und Weiterverwendung zur Verfügung**."
